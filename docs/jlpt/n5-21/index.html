应用课文#

*歌舞伎
小野：少し　急ぎましょうか。遅れない　ほうが　いいですから。
李：そしましょう。ところで，小野さんは　よく　歌舞伎を　見ますか。
小野：仕事の　後で，何度か　見に　行った　ことが　あります。終わりは　9時過ぎですね。食事は　どうしますか。
李：歌舞伎を　見た　後で，食べますか。それとも，見る　前に　何か　食べますか。
小野：そうですね・・・，休憩時間に　食事を　しましょうか。
李：ええ，いいですよ。
小野：じゃあ，早く　行って　お弁当を　買った　ほうが　いいですね。
李：言葉は　分かりませんでしたが，とても　よかったです。
小野：そうですね，課長に　感謝しましょう。
李：ええ。京劇も　いいですが，歌舞伎も　すばらしいですね。小野さんは　京劇を　見た　ことが　ありますか。
小野：いいえ，一度も　ありません。いつか　見たいですね
李：じゃあ，ぜひ　北京へ　京劇を　見た　来て　ください。

本课语法#

本课主要讲解动词“た形”的使用，“た形”的变换方式是把“て形”的“て”换成“た”，把“で”换成“だ”。(见14课)

1. 动词的“た形”#

类别ます形去掉ますて形
一类动词書きます(写)かきますかきかいた書いた
行きます(去)いきますいきいった行った
急ぎます(急)いそぎますいそぎいそいだ急いだ
飛びます(飞)とびますとびとんだ飛んだ
読みます(读)よみますよみよんだ読んだ
死にます(死)しにますしにしんだ死んだ
待ちます(等)まちますまちまった待った
売ります(卖)うりますうりうった売った
買います(买)かいますかいかった買った
話します(说)はなしますはなしはなした話した
二类动词食べます(吃)たべますたべたべた食べた
見ます(看)みますみみた見た
三类动词来ます(来)きますききたきた
します(做)しますししたした

2.语法#

見た　ことが　あります　#

和之前学过的过去式用到的“た”类似，表示经历，通常用于至少半年前发生的事情。叙述昨天等，离现在很近的以前的经历，不能使用本句型。

歌舞伎を　見た　後で，食べますか　#

用于表示一个动作在另一个动作之后，也可以用“名词+の+後で”的形式。比如

仕事の　後で，映画を　見ます。　

買った　ほうが　いいです　#

表示在两种事物中进行选择。否定句式用“ない形”替换“た形”。如果建议对方做自己想要的动作、行为时，句尾多加“よ”。

少し　急ぎましょうか　#

提议对方和自己做某件事，17课语法4学到的“~ませんか”，“~ましょう”只表示提议对方和自己做某事。而“~ましょうか”作为疑问句，还有确认提议是否可行的意思。

早く　行って　お弁当を　買った　ほうが　いいです　ね#

用来表示比较、选择。在上下文语境中，谈到对方的事情时，可用“そのほうが　い　いです”表示接受对方的提议。

仕事の　後で，何度か　見に　行った　こ　とが　あります　#

20课学到过 疑问词+か 表示不确定语义的形式。“何度か”表示不确定几次。

終わりは　9時過ぎですね。食事は　どう　しますか#

表示超过某时间或者年龄。表示年龄时用 “数词+過ぎ”，中间不加“歳”。
