新标日初上-15

语法#

本课主要讲解动词“て形”的各种句式

1. 正在进行时#

表示正在进行时，用 ”动词て形+います"

小野さんは　今　新聞を　読んで　います（小野女士现在正在看报纸）

2. 许可句式，包括请求许可和肯定回答#

肯定句式：

家で　仕事を　しても　いいです。（在家工作也可以）

疑问以及回答：

ここで　写真を　撮っても　いいですか。（这里可以拍照吗？）
いいですよ。（可以。）

回答句式的省略：

家に　帰っても　いいですか。（可以回家吗？）
はい，帰っても　いいです。（一般回答）
はい，いいです。（省略）

这种表达方式一般不用尊长，会显得傲慢

3. 表禁止做某事#

この　部屋に　入っては　いけませんか。（不能进这个房间吗？）
はい，入っては　いけません。（是的，不能进去）
いいえ，入っても　いいです。（不，可以进）

对尊长，一般不用~ては いけません，而是使用~ないで　ください。

4. 动词与某物相关，依附于某物#

用助词 に 连接，而不是 を

小野さんは　公園で　ボート　に　乗りました。（小野女士在公园里划了船。）

5. 去往目的地#

第六课学习了，助词 へ ，用于去往某地的句式。也可以用 に 表示

李さんは　病院に　行きます。（小李去了医院。）

に 和 へ 的用法区别:#

可以用に 的场景:
